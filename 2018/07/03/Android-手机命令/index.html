Android 获取手机信息命令

手机命令就是指在手机拨号界面输入某些特定的组合键，而出现的某些功能。

比如输入 *#06# 则会出现 IMEI。也叫手机串号。根据手机的不同，命令也不同，部分通用。

通用命令

命令 用途 备注
*#06# 显示IMEI号

华为手机

命令 用途 备注
*#0000# 软件版本 关于手机
##2846579## 进入工程模式

360手机

命令 用途 备注
*20121220# 进入工程模式
##4636## 统计信息 手机信息、使用情况统计
*#0000# 软件版本 关于手机
